# Веб-приложение для получения актуальной информации о погоде в любом городе мира, а также прогноза на 5 дней.

Мой проект - это одностраничное приложение (SPA), которое позволяет пользователю:

- Узнать текущую погоду в любом городе мира
- Посмотреть прогноз на 5 дней
- Автоматически определить погоду по текущему местоположению

Приложение использует открытое API OpenWeatherMap и не требует серверной части - весь код выполняется на стороне клиента.


## Функциональные возможности

| Функция | Описание |
|---------|----------|
| **Поиск по городу** | Введите название города на любом языке |
| **Геолокация** | Определите погоду в вашем текущем местоположении одним нажатием |
| **Текущая погода** | Температура, ощущаемая температура, описание, влажность, скорость ветра |
| **Прогноз на 5 дней** | Карточки с днём недели, датой, иконкой и температурой |
| **Информация о стране** | Отображение флага и полного названия страны |
| **Индикатор загрузки** | Визуальная обратная связь во время ожидания ответа от API |
| **Адаптивный дизайн** | Корректное отображение на ПК, планшетах и смартфонах |
| **Обработка ошибок** | Понятные сообщения при неверном вводе или проблемах с сетью |

## Технологии

| Технология | Назначение |
|------------|------------|
| HTML5 | Структура страницы |
| CSS3 | Стилизация, анимации, адаптивный дизайн |
| JavaScript (ES6+) | Логика приложения, работа с API, управление DOM |
| Fetch API | Асинхронные запросы к OpenWeatherMap и RestCountries |

### Используемые API

| API | URL | Назначение |
|-----|-----|------------|
| OpenWeatherMap (текущая погода) | `api.openweathermap.org/data/2.5/weather` | Получение текущей погоды |
| OpenWeatherMap (прогноз) | `api.openweathermap.org/data/2.5/forecast` | Получение прогноза на 5 дней |
| RestCountries | `restcountries.com/v3.1/alpha/{code}` | Получение флага и названия страны |
